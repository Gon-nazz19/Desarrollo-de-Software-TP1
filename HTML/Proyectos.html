Proyectos

Actualmente me encuentro como docente investigador en la UTN - Facultad Regional de San Francisco, siendo participe de grupos como GISAI y Base de Datos

Organización: Facultad de Ciencias Exactas y Naturales

Período: julio 2022 - abril 2023

Descripción: Como alumno, participé del desarrollo de una plataforma de análisis de datos académicos que utilizaba tecnologías como Python y MongoDB para procesar grandes volúmenes de datos estudiantiles. Implementé algoritmos para predecir el rendimiento académico y diseñé interfaces web con React para visualizar los resultados y ayudar a los docentes a identificar estudiantes en riesgo. El proyecto permitió a la universidad mejorar el sistema de tutorías y aumentar la retención estudiantil en un 12%.

Organización: Facultad de Ciencias de la Salud

Período: Septiembre 2023 - julio 2024

Descripción: Como investigador, Como parte de un equipo interdisciplinario, desarrollé un sistema basado en inteligencia artificial para la predicción de enfermedades cardiovasculares. Implementé redes neuronales utilizando TensorFlow y Keras, y diseñé la arquitectura backend en Python para integrar el modelo predictivo con los sistemas hospitalarios. Además, trabajé en la creación de APIs RESTful que permitían a los médicos acceder a los datos y resultados en tiempo real a través de una aplicación web intuitiva. Los resultados fueron presentados en conferencias internacionales y publicados en revistas científicas.
